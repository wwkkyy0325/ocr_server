# OCR Server - 智能文字识别工具

OCR Server 是一款基于 PaddleOCR 的高性能本地化智能文字识别工具。它专为批量处理、自动化工作流和高精度识别需求而设计，提供现代化的 GUI 界面和丰富的功能特性。

## 主要特性

*   **高精度识别引擎**: 集成 PaddleOCR v3/v5 模型，支持中英文及多语言混合识别，对文档、表格、证件等具有极高的识别准确率。
*   **强大的批量处理**:
    *   支持对文件夹内的图片进行批量导入与识别。
    *   **PDF 深度支持**: 能够将 PDF 文件作为虚拟文件夹导入，自动拆分页面并逐页识别。
*   **现代化 GUI 界面**:
    *   基于 PyQt5 构建的流畅界面。
    *   **可视化校对**: 提供卡片式结果展示，支持放大、缩小、平移查看原图与识别结果的对照。
    *   **拖拽支持**: 支持直接拖拽文件或文件夹到列表中进行处理。
*   **数据结构化与绑定**:
    *   **字段绑定**: 支持自定义字段规则，从 OCR 结果中提取关键信息（如“姓名”、“身份证号”等）并结构化输出。
    *   结果支持导出为 JSON、TXT 等格式，或直接存入数据库。
*   **GPU 加速支持**: 支持 NVIDIA GPU 加速（需配合 CUDA 运行库），在大批量任务下性能提升显著。
*   **HTTP API 服务**: 内置轻量级 HTTP Server，可作为后台服务接收外部 OCR 请求。
*   **自动化与扩展**: 内置简单的自动化与爬虫辅助模块，支持更复杂的业务流程。

## 安装说明

### 方式一：下载安装包 (Windows)

如果您是普通用户，建议直接下载最新的预编译安装包：

1.  下载并运行 `OCR_Server_Setup.exe` 进行安装。
2.  安装完成后，双击桌面上的 `OCR Server` 图标即可启动。
3.  **开启 GPU 加速 (可选)**:
    *   如果您的电脑拥有 NVIDIA 显卡，并希望获得更快的识别速度。
    *   请在安装目录下找到并运行 `install_libs.py` 脚本。
    *   该脚本会自动下载并配置所需的 CUDA 运行库（约 1GB+），配置完成后重启软件即可生效。

### 方式二：源码安装 (开发人员)

如果您是开发者或希望在其他环境运行：

#### 前置要求
*   Python 3.9+ (推荐 3.9)
*   CUDA Toolkit 11.8 (如需 GPU 加速)

#### 安装步骤

1.  **克隆代码仓库**:
    ```bash
    git clone https://github.com/your-username/ocr-server.git
    cd ocr-server
    ```

2.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **运行程序**:
    ```bash
    python run.py
    ```

## 使用指南

1.  **添加文件**:
    *   点击工具栏的“添加文件”或“添加文件夹”按钮。
    *   或者直接将图片/PDF文件拖入左侧文件列表。
2.  **开始识别**:
    *   选中列表中的文件（支持多选），右键点击“重新处理”或点击工具栏的开始按钮。
    *   对于 PDF 文件，展开节点后可单独查看每一页的识别结果。
3.  **查看与编辑**:
    *   中间区域显示原图，右侧区域显示识别到的文本卡片。
    *   在右侧卡片中可直接编辑识别错误的文字。
4.  **字段绑定 (高级)**:
    *   在菜单栏中打开“字段绑定”对话框，配置提取规则，实现结构化数据提取。

## 构建发布包

本项目包含完整的打包工具链，可生成独立的 Windows 安装包：

```bash
# 生成安装包 (需安装 Inno Setup)
python dist_tools/build_dist.py installer
```

构建产物将位于 `dist_output_installer` 目录。
